# Contributing to Lokutor JS SDK

We welcome contributions! Please follow these guidelines:

1. **Fork the repo** and create your branch from `main`.
2. **Setup your environment** by following the [Development Guide](./DEVELOPMENT.md).
3. **Make your changes**. Ensure you add tests if you're adding new functionality.
4. **Run tests** and linting to make sure everything is correct.
5. **Submit a Pull Request** with a clear description of your changes.

### Code Style
We use standard TypeScript conventions and Prettier for formatting.
